## API

### Space

| 参数      | 说明             | 类型                              | 默认值     |
| --------- | ---------------- | --------------------------------- | ---------- |
| alignment | 对齐的方式       | top \| center \| bottom           | center     |
| direction | 排列的方向       | vertical \| horizontal            | horizontal |
| size      | 间隔大小         | number \| medium \| small \| mini | -          |
| wrap      | 设置是否自动折行 | boolean                           | true       |
| spacer    | 间隔符           | string \| JSXNode                 | -          |

### Space Slot

| 名称    | 说明               |
| ------- | ------------------ |
| default | 需要添加间隔的元素 |
