## API

### Spin

| 参数           | 说明                      | 类型                    | 默认值 |
| -------------- | ------------------------- | ----------------------- | ------ |
| spinning       | 是否加载中状态，必选参数  | boolean                 | false  |
| size           | 尺寸                      | medium \| small \| mini | -      |
| delay          | 动画延迟开启时长，单位 ms | number                  | 100    |
| tip            | 自定义描述文字            | string                  |        |
| containerStyle | 外层容器的 css 样式       | styleObject             | -      |
