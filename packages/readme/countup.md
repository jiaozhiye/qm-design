## API

### Countup

| 参数     | 说明             | 类型                    | 默认值 |
| -------- | ---------------- | ----------------------- | ------ |
| endValue | 结束值，必要参数 | number                  | -      |
| size     | 尺寸             | medium \| small \| mini | -      |
| delay    | 动画延迟时长     | number                  | 0      |

### 事件

| 事件名称 | 说明               | 回调参数 |
| -------- | ------------------ | -------- |
| ready    | 计数动画开始前触发 | -        |

### 说明

可以通过标签属性 class 或 style 设置当前元素样式
